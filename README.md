# Trang Thuyet Trinh: Lich Su Dang

Du an duoc setup bang Vite + React + TypeScript, toi uu cho trang thuyet trinh 1 trang (single page) ve chu de lich su Dang.

## Tinh nang

- Bo cuc trinh chieu doc gom: Mo dau, cac moc lich su, thanh tuu, bai hoc, ket luan.
- Giao dien responsive cho desktop va mobile.
- Thiet ke co nen gradient, typo ro rang, hieu ung hien thi nhe khi tai trang.

## Cach chay du an

1. Cai dependency:

```bash
npm install
```

2. Chay dev server:

```bash
npm run dev
```

3. Build production:

```bash
npm run build
```

4. Preview ban build:

```bash
npm run preview
```

## Cau truc quan trong

- `src/App.tsx`: Noi dung va cau truc cac section trinh bay.
- `src/App.css`: Style cho bo cuc slide, timeline, danh sach.
- `src/index.css`: Theme tong, font, nen va animation toan trang.
